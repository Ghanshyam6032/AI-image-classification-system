# 🖼️ AI Image Classification System

An AI-powered multi-class image classification system built using MobileNetV2, FastAPI, Streamlit, and TensorFlow.

## 🚀 Features

* Upload images and get instant predictions
* MobileNetV2 Transfer Learning Model
* FastAPI Backend
* Streamlit Frontend
* Confidence Score Visualization
* Class Probability Chart
* Prediction History Tracking
* Render Deployment Ready

## 📂 Classes

* 👕 Clothing (Attire)
* 🎈 Decoration & Signage
* 🍔 Food
* 📦 Miscellaneous

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* MobileNetV2
* FastAPI
* Streamlit
* Pandas
* NumPy
* Pillow
* Render

## 📊 Model Information

* Architecture: MobileNetV2
* Input Size: 128 × 128
* Classes: 4
* Dataset Size: 5,983 Images
* Validation Accuracy: 73.18%
